# VShot v2 - UI 디자인 스펙

## 1. 프로젝트 개요

**VShot v2**는 실시간 영상통화 + 고해상도 사진 촬영이 가능한 포토부스 웹 애플리케이션입니다.

- **사용자**: Host (VTuber/버튜버), Guest (실사 카메라)
- **핵심 경험**: 실시간 영상 합성, 사진 촬영, 프레임 생성

---

## 2. 디자인 시스템

### 2.1 테마 컬러

| 이름 | HEX | 용도 |
|------|-----|------|
| **Primary** | `#FC712B` | 주요 버튼, 강조 요소 |
| **Secondary** | `#FD9319` | 보조 버튼, 액센트 |
| **Dark** | `#1B1612` | 텍스트, 다크 배경 |
| **Light** | `#F3E9E7` | 밝은 배경, 카드 |
| **Neutral** | `#E2D4C4` | 보더, 구분선 |

### 2.2 색상 변형

```
Primary:
├─ Default: #FC712B
├─ Light:   #FD8A4F
└─ Dark:    #E35F1F

Secondary:
├─ Default: #FD9319
├─ Light:   #FEA741
└─ Dark:    #E47F0D

Dark:
├─ Default: #1B1612
├─ 50:      #2A2420
└─ 100:     #38322E

Light:
├─ Default: #F3E9E7
├─ 50:      #FFFFFF
└─ 100:     #F9F2F0
```

### 2.3 타이포그래피

- **폰트**: 시스템 폰트 (sans-serif)
- **크기**: Tailwind 기본 스케일 사용
- **무게**: Regular (400), Medium (500), Bold (700)

### 2.4 간격 및 레이아웃

- **기본 단위**: 4px (Tailwind rem 기반)
- **컨테이너 패딩**: 16px (모바일), 24px (태블릿), 32px (데스크톱)
- **카드 패딩**: 16px ~ 24px
- **요소 간격**: 8px, 12px, 16px, 24px

│         "실시간 포토부스"            │

│                                     │

   - 방 ID 표시 + 복사 버튼
   - 연결 상태 인디케이터 (초록/회색 점)

2. **비디오 영역**
   - 2:3 종횡비 고정
   - 체크보드 배경 (투명도 표시용)
   - 카운트다운 오버레이
   - 플래시 효과 오버레이

3. **설정 패널**
   - 소스 선택 버튼 그룹
   - 토글 스위치 (크로마키)
   - 슬라이더 (민감도, 부드러움, 녹화시간, 촬영간격)
   - 프레임 레이아웃 선택 그리드

4. **썸네일 영역**
   - 8개 사진 썸네일 (1행)
   - 각 썸네일에 번호 표시
   - Guest 선택 상태 하이라이트

5. **액션 버튼**
   - 촬영 시작 (Primary)
   - 영상 생성 (Secondary)

## 6. 오버레이 및 효과

### 6.1 카운트다운 오버레이

```
┌─────────────────────┐
│                     │
│                     │
│         3           │  ← 큰 숫자 (중앙)
│                     │
│                     │
└─────────────────────┘

- 배경: rgba(0, 0, 0, 0.7)
- 숫자: 흰색, 굵게, 매우 큼 (8rem+)
- 애니메이션: scale + fade (매 초)
```

### 6.2 플래시 오버레이

```
- 배경: 흰색 (opacity: 0.9 → 0)
- Duration: 300ms
- Timing: ease-out
```

### 6.3 로딩 오버레이

```
┌─────────────────────┐
│                     │
│      ◠ 로딩...      │  ← 스피너 + 텍스트
│                     │
└─────────────────────┘

- 배경: rgba(0, 0, 0, 0.5)
- 스피너: Primary 색상
```

## 7. 애니메이션 가이드

### 7.1 기본 트랜지션

| 요소 | 속성 | Duration | Easing |
|------|------|----------|--------|
| 버튼 호버 | transform, opacity | 150ms | ease-out |
| 카드 호버 | transform, shadow | 200ms | ease-out |
| 모달 열기 | opacity, transform | 200ms | ease-out |
| 토스트 | transform | 300ms | ease-out |
| 페이지 전환 | opacity | 200ms | ease-in-out |

### 7.2 특수 애니메이션

**카운트다운**:
```css
@keyframes countdown {
  0% { transform: scale(1.2); opacity: 0; }
  20% { transform: scale(1); opacity: 1; }
  80% { transform: scale(1); opacity: 1; }
  100% { transform: scale(0.8); opacity: 0; }
}
/* Duration: 1000ms per number */
```

**플래시**:
```css
@keyframes flash {
  0% { opacity: 0.9; }
  100% { opacity: 0; }
}
/* Duration: 300ms */
```

**펄스 (연결 상태)**:
```css
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}
/* Duration: 2000ms, infinite */
```
